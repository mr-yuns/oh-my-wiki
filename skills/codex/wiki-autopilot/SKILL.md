# Wiki Autopilot

Use when the user asks to keep wiki knowledge up to date.

1. Search first with `omw wiki search "<topic>"`.
2. If reusable knowledge is missing, capture a Raw note.
3. Use ingest preview before any durable promotion.
4. Refresh the contract or index when structure or searchable content changed.
5. Keep all durable writes review-first.
