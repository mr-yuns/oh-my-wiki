# Wiki Capture

Use when the current work produced reusable knowledge.

1. Summarize only reusable, non-secret knowledge.
2. Run `omw wiki capture --type agent_session --title "<title>" --stdin`.
3. Never include credentials, signed URLs, local private paths, or raw transcripts.
4. Mention the created Raw note path.
