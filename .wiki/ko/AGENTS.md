# Base Wiki Agent Rules

이 wiki는 OMW의 공개 base wiki입니다. 에이전트는 운영 규칙과 템플릿을 유지하되, 특정 조직, 제품, 고객, 개인을 식별할 수 있는 실제 노트를 추가하지 않습니다.

## 기본 원칙

- 실제 팀 지식과 개인 메모는 공개 base wiki에 작성하지 않습니다.
- 재사용 가능한 운영 규칙, 템플릿, 지도, 카탈로그만 유지합니다.
- 새 노트를 만들기 전 `06. Resources/06-02. 지도/06-02-01. 지식 지도.md`에서 위치를 확인합니다.
- 작성 규칙은 `06. Resources/06-01. 가이드/06-01-02. 노트 작성 규칙.md`을 따릅니다.
- AI 산출물은 후보로만 취급하고 검증 상태를 명확히 표시합니다.
- 공개 전에는 민감정보, 내부 URL, 개인 경로, 토큰, 고유 제품명을 검색합니다.

## 유지해야 하는 것

- PARA 폴더 구조
- Obsidian 설정 중 공개 가능한 기본 설정
- 운영 가이드, 지도, 카탈로그
- Raw, Literature, Permanent, Project, Area, Archive 템플릿
- wiki 검증 스크립트

## 제외해야 하는 것

- 실제 조직 문서와 제품 구현 지식
- 개인별 일간 리포트와 회의록
- 고객명, 계정, 인증 정보, 내부 시스템 URL
- 사내 프로젝트 전용 인증, 이슈 추적, 대시보드 링크
- 특정 팀만 이해할 수 있는 약어와 비공개 도메인 지식
