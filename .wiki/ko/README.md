# OMW base wiki

이 wiki는 Oh My Wiki가 기본으로 제공하는 Obsidian 기반 base wiki입니다. 실제 조직 문서나 개인 노트가 아니라, 사용자가 자신의 wiki를 시작하거나 연결할 때 참고할 수 있는 운영 규칙과 템플릿만 포함합니다.

| AI 도구 연동 원칙 | `06. Resources/06-01. 가이드/06-01-04. AI 도구 연동 원칙.md` |

## 공개 패키지 경계

- 이 base wiki에는 실제 팀 문서, 고객 정보, 비공개 제품 지식, 개인 노트를 포함하지 않습니다.
- 사용자가 별도 wiki를 가지고 있으면 setup 시 해당 wiki를 연결합니다.
- 별도 wiki가 없으면 이 `.wiki`를 시작점으로 사용할 수 있습니다.
- 공개 배포 전에는 `rg`와 패키지 dry-run으로 민감한 문자열과 포함 파일을 확인합니다.
