# 지식 지도

base wiki의 최상위 탐색 시작점입니다.

## 빠른 이동

| 목적 | 문서 | 확인할 것 |
|---|---|---|
| 지식 후보 판단 | 에이전트 지식 적재 가이드 | 공개 가능성, 민감정보, 승격 위치 |
| 노트 작성 | 노트 작성 규칙 | 폴더, 파일명, frontmatter, 본문 구조 |
| 운영 검증 | Wiki 운영 절차 | 체크리스트, 검증 명령, 패키지 경계 |
| AI 산출물 정리 | AI 도구 연동 원칙 | 도구 중립성, 검증 상태 |
| Raw 수집물 관리 | Raw 노트 운영 규칙 | ingest 상태, 승격 기준 |
| 탐색 속성 관리 | Wiki 탐색 속성 카탈로그 | 분류 값과 권장 탐색 깊이 |
| 책임 영역 관리 | Wiki 영역 카탈로그 | Areas 운영과 확장 기준 |

## 공개 base wiki 정책

이 저장소의 `.wiki`는 사용자가 바로 시작할 수 있는 base wiki입니다. 실제 조직 지식은 setup 시 별도 wiki를 연결하거나, 사용자가 자신의 저장소에서 `.wiki`를 비공개로 관리합니다.
