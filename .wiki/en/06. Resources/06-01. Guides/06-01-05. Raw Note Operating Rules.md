# Raw Note Operating Rules

Raw notes are unprocessed input. The public base wiki ships empty Raw folders and templates, not real team material.

## Raw Locations

- `01-01-01. Web Clipper`: web document captures
- `01-01-02. Daily Reports`: daily observations
- `01-01-03. Agent Sessions`: agent work sessions
- `01-01-04. Discussions`: meetings and discussions

## Ingest States

Use `captured`, `reviewing`, `promoted`, `held`, `discarded`, or `archived`.
