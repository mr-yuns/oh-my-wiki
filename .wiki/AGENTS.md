# Base Wiki Agent Rules

This public base wiki provides language-specific starter content under `en/` and `ko/`. Agents should keep both language roots free of private team, customer, product, credential, and personal material.

- Use `--language en` or `--language ko` to choose the active base wiki contract.
- Keep root-level scripts and Obsidian defaults shared.
- Keep real user knowledge in the selected user wiki, not in the public base wiki package.
